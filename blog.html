The scope of var, let, and const...

*Var: It is a function scoped. It can be used anywhere from the function. If var declared outside function, it will be accessible for entire program.


*Let: Variable thats are declared with 'let', they will accessible only specific block. So, it is block scoped. 'let' are not hoisted to the top.


*Const: It is also block scoped. Variable declared with 'const' cannot be reassigned

The use cases of null and undefined...

'null' and 'undefined' both of them are primitive value in JavaScript. The 'null' arise to represent the absence of a value. And 'undefined' arise from where JavaScript don't know the value of something.

REST API

REST(representational State Transfer Application Programming Interface. It is combination of rules and conventions, that helps to build and interact with web services. It is an architectural style protocol.)
